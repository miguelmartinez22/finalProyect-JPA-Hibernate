¿QUIÉNES SOMOS?


Este proyecto es desarrollado por dos estudiantes de Desarrollo de Aplicaciones Web. Surge como proyecto de la segunda evaluación del módulo de Desarrollo Web en Entorno Servidor de ambos programadores.

Como modelo tomamos el proyecto del módulo de bases de datos de Javier Martinez realizado en el curso anterior.

El proyecto se basa en la creación de un sitio web que promociona entradas para un festival. Dicho proyecto es elaborado, principalmente, en Java. Se emplean, también, otros componentes de diseño y maquetación web: HTML5, CSS3, Bootstrap, Thymeleaf como motor de plantillas y Spring Boot. También recurrimos a una base de datos MySQL que se encuentra en un servicio Docker.

Esta página tiene varias vistas:
La primera es la del usuario que no se da de alta, al que se le permite navegar por la web y comprar una entrada para el festival.
Existe otra vista que es la del cliente, pues ya se ha dado de alta en el sistema al comprar una entrada y puede ver los datos de la misma.
La tercera y última vista es la de administrador. Se trata de un usuario capaz de gestionar la página añadiendo nuevos artistas y asignarles un nuevo camerino, nuevos carteles, los clubs de fans, etc.
